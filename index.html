Gemini API キーの設定

AI機能を利用するには、Google AI Studio で発行した Gemini API キーが必要です。
キーはブラウザのローカルストレージに保存され、外部には送信されません（Gemini API 呼び出し時を除く）。

出力設定

Googleスプレッドシートへ反映する場合に使用する、Google Apps Script WebアプリURLとシート名を設定します。設定値はブラウザのローカルストレージに保存されます。
